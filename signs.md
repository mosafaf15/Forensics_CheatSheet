
# file signatures

### JPEG
- start : `FF D8 FF E0` or `FF D8 FF E1`
- end sign : `FF D9`

### PNG
- start : `89 50 4E 47 0D 0A 1A 0A`

### PDF
- start : `25 50 44 46`
- end : `25 25 4F 46`

### ZIP
- start : `50 4B 03 04`
- end : `50 4B 05 06`

### RAR
- start : `52 61 72 21 1A 07 00 00`
- end : `52 61 72 21 1A 07 00 00`

### ISO
- start : `43 44 30 30 02 01 01 00`

### EXE
- start : `4D 5A`

### WAV
> RIFF
- start: `52 49 46 46`
> WAVE
- start: `57 41 56 45`
